Configurar Telegram
Siga os passos para ativar o bot no Telegram
✈️ SETUP DO BOT
1
Abra o Telegram e fale com @BotFather. Envie /newbot e siga as instruções para criar seu bot.
2
Copie o token gerado (formato: 1234567890:ABC...) e adicione ao arquivo .env como TELEGRAM_TOKEN=SEU_TOKEN
3
No Railway, adicione as variáveis de ambiente: TELEGRAM_TOKEN, ANTHROPIC_API_KEY e APP_URL (a URL do seu deploy).
4
Faça o deploy. O webhook é configurado automaticamente. Envie /start para seu bot e ele responderá!
5
Para desenvolvimento local, defina DEV_MODE=true no .env. O bot usará polling em vez de webhook.
